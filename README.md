# PS-IC: Perturbation Subsampling for Large-Scale Interval-Censored Data

This repository provides the Julia code for the perturbation subsampling method for the proportional hazards model with intervalвЂ‘censored data, described in the paper:

> **Perturbation Subsampling for Large-Scale Interval-Censored Data**

- `simulation/code/` вЂ“ Scripts to run and summarize the simulation study.

## Reproducing the Simulation Results

### 1. Run a single simulation replicate

Each call generates one replicate of the simulation.  
The simulation setting (i), (ii), or (iii) is controlled by the constant `h_type` at the top of `simulation.jl`.  
Set `h_type = 1` for setting (i), `h_type = 2` for setting (ii), or `h_type = 3` for setting (iii) before running.

From the `simulation/code/` directory, execute:

```bash
julia simulation.jl <sim_id> <n_sim> <n_sims>
```

- `sim_id` : process index (в‰ҐвЂЇ1), used for seeding  
- `n_sim` : replicate index within the process (1вЂ‘based)  
- `n_sims`: total number of replicates per process  

The script saves a `.jld2` file containing all metrics for that replicate into the corresponding results folder (e.g., `simulation/results/simu_hz1_500/` for setting (i)).

Parallel execution can be launched via the `run_parallel.sh` script in the same directory.

### 2. Aggregate results

After collecting all replicates, run the aggregation script from the `simulation/code/` directory:

```bash
julia simulation_summary.jl
```

It reads all `simulation_results_*.jld2` files in each setting folder and produces aggregated result files (`aggregated_results_<N>.jld2`) containing average runtimes, MSEs, and coverage probabilities.  The script also prints coverage tables matching those in the paper.
